WMS Configuration

The WMS provides functionality to add map using service URL.

Here is the explanation of how to add, configure and remove WMS map in Hale studio. To work with WMS maps, first open Map perspective using Open Perspective tool menu reside in the right corner of the studio.

To add the WMS maps in to Map perspective, click on the View Menu button and then heads to WMS maps→Add.

By clicking on Add button, a dialog will be appeared, which asks for the Name and Service URL. A user have to insert a unique name and URL from where map tiles will be fetched.

Please note here name must be unique from any other WMS map configurations and URL should be valid to fetch capabilities. To go to the next configuration page, click on Next button in the dialog or click Finish directly to skip next configuration pages.

On the next page, select Spatial reference system supported by the WMS. expand the combo and select the respective SRS. To go to the next configuration page, click on Next button in the dialog or click Finish to skip next configuration pages and complete configuration.

On this page, select Layers supported by the WMS. Click on the Change button to select layers of the maps. Here you can select multiple layers by clicking on the check box. To go to the next configuration page, click on Next button in the dialog or click Finish to skip next configuration pages and complete configuration.

On this page, configure map Tiles. You have to define the maximum zoom level for the map, minimum tile size(px) as width or height and minimum map size(px) as width or height. Please note, you have to define values between 1 to 100 for zoom levels, between 64 to 1024 for tile size and between 64 to 65,536 for map size otherwise you can not proceed to finish. Click Finish to complete the configuration.

After finishing configuration, WMS maps will be loaded to Map perspective and also added in to the view menu.

You can again reconfigure the map added by you, by clicking on the name you configured before. It will reopen the same dialogs as per above in same order. You can change the configurations in those dialogs.
